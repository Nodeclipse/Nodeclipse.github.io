eclipse开发nodejs介绍

 

准备环境

1.安装nodejs环境，下载地址：http://nodejs.org/

2.eclipse选用的是aptana最新的版本，下载地址：http://www.aptana.com/

 

安装插件

aptana安装nodejs和coffee的插件。请参见：http://www.nodeclipse.org/

具体请咨询保罗

设置界面

使用aptana自带的Terminal，此工具是git的操作命令行，支持少量的shell命令。

可以使用他来运行nodejs或coffee。

如图：

 

输入cd /e/ 命令后就直接进入到了windows的e：盘

我们使用命令来安装coffee

npm install coffee

安装后就可以使用coffee开发nodejs代码了。

设置 tab policy：这个设置很重要，否则开发将危险重重。

window -> Preferences
